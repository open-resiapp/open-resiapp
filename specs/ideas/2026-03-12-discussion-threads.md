## Goal
Informal group discussions about building/entrance topics with optional
lightweight polls. Separate from formal voting (legally binding) and
1:1 private messages.

## Scope

### In scope
- Threaded discussion topics (per building or per entrance)
- Group chat style messages within a thread
- Optional poll attached to a thread (e.g. "Modrá / Zelená / Biela")
- Scoping: building-wide or per entrance

### Out of scope
- Legally binding voting (that's the existing voting system)
- 1:1 private messages (that's RES-20260312-002)
- File attachments beyond photos (MVP)

## Acceptance Criteria


## Project Context
Owners need a place to discuss building topics informally before
formal votes happen. Polls give a quick temperature check without
the legal overhead of formal voting (§14a). This fills the gap
between private messages and official voting.
